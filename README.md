# Challenge Alura Store

Este proyecto es un análisis de datos diseñado para evaluar el desempeño de múltiples sucursales de una cadena de tiendas. El objetivo principal es identificar patrones de ventas, costos logísticos y satisfacción del cliente para tomar decisiones estratégicas basadas en datos.

## Objetivo del Proyecto

El dueño de la cadena de tiendas, el "Sr. Juan", necesita optimizar sus operaciones. El análisis busca responder a la pregunta estratégica: **¿Qué tienda debería cerrarse o venderse debido a su bajo rendimiento?**

Para ello, se analizaron 4 grandes datasets con información sobre:
* Ingresos totales por ubicación.
* Productos más y menos vendidos.
* Costos de envío promedio.
* Calificación y satisfacción del cliente.

## Tecnologías Utilizadas

* **Python**: Lenguaje principal.
* **Pandas**: Para la manipulación y limpieza de datos (ETL).
* **Matplotlib & Seaborn**: Para la visualización de datos y generación de gráficos.
* **Google Colab**: Entorno de desarrollo.

## Metodología

1.  **Carga y Unificación**: Se integraron 4 datasets dispersos (uno por tienda) en un único DataFrame maestro.
2.  **Limpieza de Datos**: Tratamiento de variables, corrección de tipos de datos (fechas).
3.  **Análisis Exploratorio**:
    * Cálculo de ingresos totales.
    * Identificación de productos "Estrella" y "Rezagados".
    * Análisis de costos logísticos.
4.  **Visualización**: Generación de gráficos de barras, líneas de tendencia y comparativas.
